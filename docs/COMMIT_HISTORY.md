# VideoTracker 提交历史与开发记录

本文档用于记录开发过程中的 Git Commit 历史与各个阶段的功能点实现。

## 提交历史日志 (Git Commit Logs)

*   **feat: 支持边看边修改当前字幕文本**
    *   **时间**: 2026-07-03
    *   **描述**: 新增右侧字幕校对面板，支持本地 `.srt` / `.vtt` 字幕按当前播放时间定位正在显示的字幕句，直接编辑文本并写回原字幕文件；主进程新增字幕读写 IPC 与一次性 `.bak` 备份保护，保存后播放器自动重新加载字幕。同步补充字幕解析、写回与文件备份测试，并发布版本号至 `v1.1.8`。

*   **fix: 修复自动更新安装包资产名不一致导致下载失败**
    *   **时间**: 2026-07-02
    *   **描述**: 修复 GitHub Actions 发布流程中 electron-builder 与 action-gh-release 双重发布造成的资产名错位问题，固定 Windows 安装包与便携版产物为无空格连字符命名，并禁用打包阶段的隐式发布，确保 `latest.yml` 指向的安装包文件真实存在。

*   **chore: 发布 v1.1.5**
    *   **时间**: 2026-07-02
    *   **描述**: 发布同目录字幕自动识别与字幕开关功能。播放器会自动挂载同名或语言后缀字幕，用户可手动打开/关闭字幕，推送 `v1.1.5` 标签触发 GitHub Actions 自动打包。

*   **feat: 支持同目录字幕自动识别与开关**
    *   **时间**: 2026-07-02

*   **chore: 发布 v1.1.4**

*   **feat: 支持外部字幕样式调整与手动更新下载控制**

*   **fix: 关闭主窗口 webSecurity 解决第三方网盘 WebDAV 连接时的 CORS 跨域限制** (b661d23)
